# Chatbot Using LangChain & LLMs

This repository contains a Jupyter Notebook (chatbot.ipynb) that demonstrates the implementation of a conversational AI chatbot using LangChain and Large Language Models (LLMs). The notebook shows how to load models, create prompt templates, build conversation chains, and interact with an LLM in a simple and modular way.
Features

LangChain-powered conversation pipeline
Custom prompt templates
LLM integration (OpenAI / Hugging Face / local models)
Memory-enabled conversation
Fully documented Jupyter Notebook
Extendable for advanced chatbot applications
